                        Model Change Bulletin

                                MCB#3                       06/30/2011

                        MAKEMET (dated 16216)

This Model Change Bulletin (MCB) documents changes made to the MAKEMET code.
A brief description of the changes is provided below. 


This revised version of MAKEMET (dated 16216) includes the following
modifications relative to version 15181:

Bug Fixes

1.  Modified the MAKEMET date variable to match latest version

Miscellaneous

1.  Modified subroutine UCALST to include the u* adjustment
    in AERMET 16216 subroutine UCALST.  The user has
    the option to invoke the adjustment.
